| Submodel                  | 对应 xacro 模块                      | 说明                       |
| ------------------------- | -------------------------------- | ------------------------ |
| **StructureSubmodel**     | igus_rebel/                      | 机械结构，link/joint 参数       |
| **ControlSubmodel**       | igus_rebel.control.xacro         | 控制接口（ros2_control、关节命令等） |
| **KinematicsSubmodel**    | igus_rebel_mod.* / urdf joint 数据 | 运动学参数（DH、限位、速度等）         |
| **VisualizationSubmodel** | 所有 urdf/xacro 文件路径               | RViz / 仿真显示资源            |
| **SafetySubmodel**        | 数据手册参数                           | 安全限制（负载、碰撞、速度）           |
| **MountSubmodel**         | mounts/mount_v1,v2.xacro         | 安装方式、固定法兰、姿态             |
| **CameraSubmodel**        | cameras/*.xacro                  | 相机与虚拟相机坐标系统              |
| **EndEffectorSubmodel**   | end_effectors/*.xacro            | 末端执行器（夹爪、触探器）            |
| **MobileBaseSubmodel**    | mobile_robot/castle.urdf         | 底盘塔体、传感器、电池模块            |
